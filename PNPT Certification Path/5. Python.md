#### Sockets

#sockets

- Used to make a connection between ports and IP addresses
We created a netcat script
```
#!/bin/python3                               
import socket
HOST = '127.0.0.1'
PORT = 7777
s = socket.socket(socket.AF_INET, socket.SOCK_STREAM) #af _inet is ipv4, sock_stream is a port

s.connect(HOST,PORT)

```
